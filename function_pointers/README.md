This file outlines what each file within the function_pointers directory does in relevance to function pointers

0-print_name.c is a function that prints a name

1-array_iterator.c is a function that executes a function given as a parameter on each element of an array

2-int_index.c is a function that searches for an integer

3-main.c, 3-op_functions.c, 3-get_op_func.c, 3-calc.h are part of a program that perform simple operations, such as adding, subtracting, multiplying, dividing and modulo
