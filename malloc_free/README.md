this README outlines what each function does, utilising the malloc function

0-create_array.c is a function that creates an array of characters, and initializes it with a specific character

1-strdup.c is a function that returns a pointer to a newly allocated space in memory.

2-str_concat.c is a function that concatenates two strings

3-alloc_grid.c is a function that returns a pointer to a 2 dimensional array of integers

4-free_grid.c is a function that frees a 2 dimensional grid previously created by the alloc_grid function
